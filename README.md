# Scripts of Test Automation
This repository contains automation test scripts developed using **Java**, **Selenium WebDriver**, **Maven**, and **BDD with Cucumber**.

## Project Structure
The codebase is organized into the following components:

### Feature Files
Contains business-readable test scenarios written in Gherkin language.

### Step Definitions
Contains the implementation of the steps defined in the feature files.

### Hooks
Contains setup and teardown methods for browser initialization and closure using Cucumber Hooks.

### Selenium Automation Scripts
Contains web automation scripts developed using Selenium WebDriver.
